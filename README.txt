Home Connect (Local)

Control and monitor your Bosch and Siemens home appliances - ovens,
dishwashers, washing machines, dryers, coffee machines and more - entirely
over your local network.

Unlike the official integration, this app talks directly to the appliance
on your LAN. After a one-time login to retrieve the per-appliance security
keys, no cloud connection is needed and you can block the appliances from
the internet entirely.

23 May 2026: 

Added sthe following features (still needs to be tested):
- **Ovens**:
  - Real-time monitoring: operation state, door state, remaining
    program time, program progress, cavity & meat probe temperatures.
  - Active microwave power level, current program name, and active
    physical interior lighting status.
  - Warning sensors for safety events like cavity temperature too high
    or software updates available.
  - Flow triggers for when preheating has completed (regular and fast
    preheat).
  - Program control and temperature setting where allowed by the
    appliance.
- **Dishwashers**:
  - Real-time monitoring: operation state, door state, remaining time
    (including estimated remaining time), program progress.
  - Active options tracking (half load, intensive zone, vario speed,
    etc.).
  - Maintenance & alert sensors: salt low status, rinse aid low status,
    remaining runs until Machine Care recommendation, filter cleaning
    checks, and drain pump blockages.
  - Software updates available alert sensor.
  - Program start, pause, resume, stop, and settings control.
- **Flow cards**:
  - Complete trigger, condition, and action cards for ovens and
    dishwashers to automate all available states, events, and settings.

Setup requires a one-time SingleKey ID (Bosch/Siemens account) login during
pairing to fetch the local keys. After that the app runs fully local.
